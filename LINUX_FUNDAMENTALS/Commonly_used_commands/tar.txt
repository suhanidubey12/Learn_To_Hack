The 'tar' command
The tar command is used to create, list, and extract archive files in Linux. 
An archive is a single file that contains multiple files and directories bundled together.

Flags:
1. -c : creates a new archive file
2. -x : extract files from an existing archive
3. -f : specifies the name of the archive file
4. -v : displays the file that is being processed
5. -z : compresses the archive
