Command: ip

Description
The ip command is used to view and manage network configuration in Linux. It replaces older tools like ifconfig, route, and arp. This command is essential for networking, troubleshooting, and CTF challenges involving network analysis.

Syntax:
ip [OPTIONS] OBJECT COMMAND


Common Objects
addr or a → Manage IP addresses
link or l → Manage network interfaces
route or r → Manage routing table
neigh → Manage ARP cache



Important Flags and Commands

1. ip addr

	- Shows all IP addresses assigned to network interfaces.

2. ip addr show

	- Displays detailed IP information for all interfaces.

3. ip link show

	- Lists all network interfaces and their status (UP or DOWN).

4. ip link set
ip link set eth0 up
ip link set eth0 down

	- Enables or disables a network interface.

5. ip route

	- Displays the system routing table. Useful for understanding network paths.

6. ip route add

ip route add 192.168.1.0/24 via 192.168.1.1

	- Adds a new route to the routing table.

7.ip neigh

	- Shows ARP table entries mapping IP addresses to MAC addresses.


Why it is useful in CTFs
	Identify network interfaces and IPs

	Detect misconfigured networks

	Analyze routes and gateways

	Useful in pivoting and lateral movement scenario
